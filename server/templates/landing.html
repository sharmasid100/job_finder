About This Project

Jinger is a job recommendation platform powered by machine learning (AdaBoost Classifier) and real-time data via JSearch API. It analyzes user preferences, technical skills, domain choices, and experience to predict the most suitable job role. Once predicted, it fetches matching job listings from platforms like LinkedIn and ZipRecruiter using RapidAPI integration.

The system uses preprocessing techniques like Label Encoding (LE) and One-Hot Encoding (OHE) to format user data. It‚Äôs built on Flask, scikit-learn, and modern HTML/CSS, and deployable on both local and cloud environments.
